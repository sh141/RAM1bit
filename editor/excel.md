# Excelの神

## 全員のアドレス一覧から参加者名を使ってアドレスを取得する

こんな状態が `all_list`シートにあったとして、

別のシートに参加者名だけのリストがあって、

参加者のアドレスを対応表から引っ張りたい、を実現する。

```excel
=VLOOKUP(A2,all_list!A:B,2,"")
```

該当ない場合は#N/Aが返ってくる

## 皆のアドレスをカンマ区切りで結合する

C列の1〜47までにあるアドレスを `,`+ `Space`区切りで結合する場合

```excel
=TEXTJOIN(", ",true,C1:C47)
```

※ trueだと空のセルを無視してくれる
